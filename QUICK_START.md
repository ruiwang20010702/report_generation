# 🚀 51Talk学习分析系统 - 5分钟快速上手

## 第一步：启动应用

打开终端，运行：

```bash
cd /Users/ruiwang/Desktop/test
npm run dev
```

看到类似信息表示启动成功：
```
  VITE v5.4.19  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## 第二步：打开浏览器

访问：**http://localhost:5173**

## 第三步：使用"快速测试"功能

1. 在表单页面，点击右上角的 **"快速测试"** 按钮
2. 表单会自动填充示例数据：
   - 学生姓名：小明
   - 年级：小学三年级
   - 级别：Level 3
   - 单元：5
   - 两个示例视频链接

## 第四步：生成报告

1. 点击底部的 **"生成学习报告"** 按钮
2. 观看加载动画（约8秒）：
   - 可爱的猴子跳动
   - 进度条增长
   - 4个分析步骤依次高亮

## 第五步：查看报告

报告自动展示，包含：

### 📊 关键学习数据
- 举手次数：↑ 15%
- 回答长度：↑ 23%
- 完整句子率：↑ 18%
- 阅读准确率：92%

### 🏆 四大维度进步分析
- 流利度
- 自信度
- 语言应用
- 句型复杂度

### 🎯 待提升点详细分析
- 发音准确性（含具体单词示例）
- 语法细节（含错误纠正）
- 语调与节奏（含改进建议）

## 第六步：下载PDF报告

1. 滚动到报告底部
2. 点击 **"下载PDF报告"** 按钮
3. 等待3-5秒，PDF自动下载
4. 文件名格式：`51Talk学习报告_小明_2025-11-5.pdf`

## 🎯 手动测试其他功能

### 测试表单验证

1. 点击左上角的 **"返回"** 按钮
2. 清空所有字段
3. 直接点击"生成学习报告"
4. 观察错误提示（红色文字）

### 测试URL验证

1. 在视频链接输入框输入无效URL：`not-a-url`
2. 观察错误提示：❌ "请输入有效的视频链接"
3. 输入有效URL：`https://www.youtube.com/watch?v=test`
4. 观察成功提示：✅ "有效的URL" + 预览链接

## 🐛 遇到问题？

### 问题1：页面空白
- 检查浏览器控制台（F12）是否有错误
- 确保 npm run dev 正在运行
- 刷新页面（Ctrl/Cmd + R）

### 问题2：PDF下载失败
- 检查浏览器是否允许下载
- 关闭广告拦截器
- 尝试使用Chrome浏览器

### 问题3：样式显示异常
- 清除浏览器缓存
- 硬刷新（Ctrl/Cmd + Shift + R）
