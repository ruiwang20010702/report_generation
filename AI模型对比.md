## 🔷 DeepSeek - 性价比之王

### 基本信息
- **提供商**：深度求索（国内）
- **官网**：https://platform.deepseek.com/
- **模型**：deepseek-chat
- **API 兼容**：完全兼容 OpenAI SDK

### 💰 定价
```
输入：¥1 / 1M tokens
输出：¥2 / 1M tokens

实际成本示例（分析 1 个视频，约 5000 tokens）：
- 成本：¥0.01（1 分钱）
- 月分析 1000 次：¥10
- 年分析 12000 次：¥120
```

### ✨ 特点
- ✅ **超高性价比**：比 GPT-4 便宜 95%
- ✅ **强推理能力**：尤其擅长逻辑分析、数学推理
- ✅ **国内直连**：无需 VPN，响应速度快
- ✅ **免费额度**：新用户送 500万 tokens（约 5000 次分析）
- ✅ **长文本支持**：支持 32K 上下文

### 📝 配置方法
```bash
# 1. 注册账号
访问 https://platform.deepseek.com/
支持微信/手机号注册（秒级完成）

# 2. 获取 API Key
控制台 → API Keys → 创建新密钥

# 3. 添加到 .env
DEEPSEEK_API_KEY=sk-你的密钥
```

### 🎯 适用场景
- ✅ 预算有限，追求极致性价比
- ✅ 需要大量调用（重度使用）
- ✅ 复杂的分析和推理任务
- ✅ 长期使用，追求低成本

### 测试命令
```bash
npm run test:deepseek
```

## 🧠 智谱 GLM-4 - 质量标杆

### 基本信息
- **提供商**：智谱AI（清华背景）
- **官网**：https://open.bigmodel.cn/
- **模型**：glm-4-plus / glm-4-flash
- **API 兼容**：完全兼容 OpenAI SDK

### 💰 定价
```
GLM-4-Plus（最强版）：
- 输入/输出：¥50 / 1M tokens

GLM-4-Flash（经济版）：
- 完全免费（有限速）

新用户福利：
- 送 25 元体验金
- 体验金可使用 GLM-4-Plus
```

### ✨ 特点
- ✅ **清华技术背景**：国内老牌大模型
- ✅ **高准确率**：输出质量稳定可靠
- ✅ **多模态能力**：支持图像理解
- ✅ **免费版本**：GLM-4-Flash 完全免费
- ✅ **长上下文**：支持 128K 上下文

### 📝 配置方法
```bash
# 1. 注册账号
访问 https://open.bigmodel.cn/
手机号/邮箱注册

# 2. 创建 API Key
控制台 → API Keys → 创建新密钥

# 3. 添加到 .env
GLM_API_KEY=你的密钥

# 可选：使用免费的 Flash 版本
# 修改代码中的模型名为 'glm-4-flash'
```

### 🎯 适用场景
- ✅ 追求高质量输出，对准确率要求高
- ✅ 需要稳定可靠的商业服务
- ✅ 预算充足，可以接受较高成本
- ✅ 需要多模态能力（图像分析等）

### 💡 成本优化建议
- 日常使用 `glm-4-flash`（免费）
- 关键场景使用 `glm-4-plus`（高质量）

### 测试命令

### Q1: 我应该选择哪个模型？

**A**: 可以！系统会按优先级自动选择，也可以随时切换。

### Q3: 免费额度用完了怎么办？

**A**:

### Q4: 模型质量有差别吗？
