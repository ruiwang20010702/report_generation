# 🚀 快速启动指南

## 前置条件

✅ Node.js 16+ 和 npm 已安装
✅ 环境变量已配置（可选，用于真实AI分析）

## 一键启动

```bash
# 同时启动前端和后端
npm run dev:all
```

然后访问: http://localhost:8080

## 快速测试（3步）

### 1️⃣ 生成报告

1. 填写表单信息
2. 保持"使用模拟数据"勾选（快速测试）
3. 点击"开始分析"

### 2️⃣ 查看报告

1. 等待分析完成（约3-5秒）
2. 查看生成的学习分析报告
3. 浏览各项分析指标

### 3️⃣ 导出报告

1. 点击底部的"下载长图报告"按钮
2. 等待长图生成完成
3. 报告会自动下载为PNG图片

## 验证成功

✅ 报告页面正常显示
✅ 所有数据指标可见
✅ 长图导出功能正常
✅ 文件名格式正确：`51Talk学习报告_姓名_日期.png`

## 故障排除

### 问题：无法连接到服务器
**解决**: 
- 确保后端服务正在运行：`npm run dev:server`
- 检查端口3001是否被占用

### 问题：长图生成失败
**解决**: 
- 确保报告页面完全加载
- 检查浏览器控制台错误信息
- 尝试刷新页面后重新生成

### 问题：AI分析失败
**解决**:
- 检查环境变量配置是否正确
- 确认API Key有效
- 查看后端日志获取详细错误信息
