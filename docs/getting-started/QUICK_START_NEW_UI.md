## 🐛 故障排除

### 如果看不到新样式
1. **清除浏览器缓存**
   - Chrome: Ctrl+Shift+Delete (Windows) 或 Cmd+Shift+Delete (Mac)
   - 或者使用无痕模式：Ctrl+Shift+N (Windows) 或 Cmd+Shift+N (Mac)

2. **硬刷新页面**
   - Chrome/Firefox: Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)

### 如果图片显示不出来
1. 检查图片文件是否存在于 `src/assets/` 目录
2. 检查浏览器控制台是否有 404 错误
3. 确认服务器正在运行

### 如果日期选择器不工作
1. 检查浏览器控制台是否有错误
2. 确认 `date-fns` 包已安装（应该已经安装了）
3. 尝试重启开发服务器
